1. 路径

1.1 得到所在路径pwd

1.2 移动路径cd

命令 说明
cd 路径 移动到指定路径
cd / 移动到根目录
cd - 移动到上次所在路径
cd . . 移动到上层目录

例如：

cd /home

2 查看

2.1 查看目录下的文件与文件夹 ls

参数 说明
-l 详细信息
-a 显示隐藏文件
-F 区别文件夹与文件（文件夹后加“/”）
-h 以合适的单位显示

例如：

cd /etc
ls -lh

2.2 查看文件内容 cat

例如：

cat /etc/hdparm.conf

2.3 查看文件内容 more

快捷键 说明
enter 下一行
空格 下一页
b 上一页
q 退出

例如：

vi a.py

for i in range(1000):
	print('-'*20,i,'-'*10)

python3 a.py >> b.txt
more b.txt

注：>和>>

  • > 和>>是把控制台中输出的内容写入指定的文件
  • > 　 如果目标文件中存在内容，会覆盖掉原内容
  • >>    如果目标文件中存在内容，会追加写入

2.4 查看文件开头 head

格式

  • head -n 【行数】 【文件】

例如：查看开头10行

head -n 10 b.txt

2.5 查看文件结尾与监听文件变化 tail

参数 说明
-n 查看结尾
-f 监听文件变化

例如：
查看文件结尾10行

tail -n 10 b.txt

监听文件变化

tail -f b.txt

3 创建

3.1 创建文件夹 mkdir

参数 说明
-p 创建多层目录

例如：

mkdir mydir
mkdir mydir2/mydir3 -p

3.2 创建文件 touch

例如：

touch a.py

4 删除 rm

参数 说明
-r 递归（用于删除文件夹以及内部的所有文件）
-i 提示：是否删除
-f 不提示

例如：删除当前目录中的所有文件

rm -rf ./*

5 移动 mv

格式：

  • mv 【原文件路径】 【目标路径】

例如：

mv a.py /home/

注：mv可以用于重命名

cd /home
mv a.py b.py

6 复制 cp

格式：

  • cp 【原文件路径】 【目标路径】

7 查找 find

参数 说明
-name 通过名字查找
-size 通过尺寸查找

模糊查询

参数 说明
* 匹配任意个任意字符
? 匹配一个任意字符

指定大小

参数 说明
+ 大于
- 小于

例如：
1.全盘查找以new开头的文件

find -name ‘new*’

2.在/etc下查找以new开头的文件

find /etc -name ‘new*’

3.查找/etc下第一个字符任意，接下来是ew，后面的字符和个数都任意的文件（正则： .{1}ew.*）

find /etc -name ‘?ew*’

4.查找/etc下大于100k的文件

find /etc -size +100k

5.查找/etc下小于100k的文件

find /etc -size -100k

8 查看路径 which

例如：

which python3

9 链接 ln

创建软连接

  • ln -s 【源文件】 【链接文件】

创建硬链接

  • ln 【源文件】 【链接文件】

软连接与硬链接的联系与区别

  1. 软连接创建的链接文件几乎不占用空间，硬链接创建的链接文件占用与源文件相同大小的空间。
  2. 软连接删除源文件后，链接文件不可用，硬链接删除源文件，链接文件依然可用。
  3. 更改源文件或链接文件中任意一方，另一方都会发生改变。

10 打包与解包

10.1 tar.gz

打包：

  • tar -zcvf 【包名.tar.gz】 【目录路径】

解包：

  • tar -zxvf 【包名.tar.gz】
参数 解释
z gz格式
c create 创建
v view 显示详细信息
f file 文件
x execute 执行

10.2 bz2

打包：

  • tar -jcvf 【包名.bz2】 【目录路径】

解包：

  • tar -jxvf 【包名.bz2】

注：唯一与tar.gz不同的地方，就是把z替换成了j

10.3 zip

安装zip

apt update
sudo apt install zip

打包

  • zip -r 【包名.zip】 【目录路径】

解包

  • unzip 【包名.zip】

10.4 查看压缩包的压缩格式 file

  • file 【压缩包文件路径】

注：如果压缩文件的扩展名是未知的，不知道用何种方式解包，就需要用到file命名
