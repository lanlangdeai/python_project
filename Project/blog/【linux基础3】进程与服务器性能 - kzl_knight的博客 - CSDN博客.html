3 查看服务器性能 top

命令 说明
M 内存排序
P CPU排序
q 退出

其他关键字解释：

关键字 说明
PR 进程优先级
NI nice值，负值表示高优先级，正值表示低优先级
VIRT 进程使用的虚拟内存总量（单位kb），VIRT=SWAP + RES
RES 进程使用的、未被换出的物理内存大小（单位kb），RES= CODE + DATA
SHR 共享内存大小（单位kb）
S 进程状态
%CPU cpu占用百分比
%MEM 内存占用百分比
TIME+ 进程使用的CPU时间总计，单位1%秒
COMMAND 进程名称（命令名/命令行）
